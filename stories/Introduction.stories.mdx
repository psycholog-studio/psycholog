# psycholog

Toki's components library.

## How to use psycholog ui

1. Install psycholog ui , `yarn add @tokeleecy/psycholog-ui`

```
# npm
# .npmrc

//npm.pkg.github.com/:_authToken="<authToken>"
@tokileecy:registry=https://npm.pkg.github.com
```

```
# yarn berry
# ~/.yarnrc.yml

npmRegistries:
  "https://npm.pkg.github.com":
    npmAuthToken: <authToken>

npmScopes:
  tokileecy:
    npmRegistryServer: "https://npm.pkg.github.com"
    npmAuthToken: <authToken>
```

2. add css reset on entry point

```js
// import reset.css
import '@psycholog-studio/ui/reset.css'
```

3. import ui component

   s

```js
// example
// eslint-disable-next-line no-unused-vars
import Input from '@psycholog-studio/ui/Input'

// ...
```
